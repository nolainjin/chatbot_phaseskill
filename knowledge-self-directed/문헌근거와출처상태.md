# 자기주도학습 논문 출처와 검증상태

이 문서는 자기주도학습 지식팩이 어떤 연구 메모를 바탕으로 만들어졌는지와, 아직
어디까지 검증되었는지를 구분하기 위한 출처 지도다. 출처 목록에 있다고 해서
자동으로 원문 검증이 끝난 것은 아니다.

## 검증 상태

- `AI활용_자기주도학습_프로그램_프레임워크_sources.md`는 문서 자체가 후보 제시용이며
  검증이 아니라고 명시한다. 해당 목록의 후보 DOI를 확정 인용처럼 말하지 않는다.
- `2026-06-30_learning-methods-*.md`의 DOI·URL은 문헌 수집·보강 목록이다. 원문과
  연구 설계, 표본, 효과크기를 확인하기 전에는 "연구가 증명했다"고 단정하지 않는다.
- 이 팩의 일반 설명은 연구 메모에서 공통으로 제시된 학습 과정 원리를 코칭 문장으로
  재구성한 것이다. 개별 학생에게 효과를 보장하는 처방이 아니다.

## 챗봇이 연구 근거를 답하는 방식

1. 먼저 개념을 설명한다.
2. 어떤 작업공간 문서에 근거했는지 밝힌다.
3. 특정 논문·효과크기를 요구하면 출처 상태를 확인 필요로 표시한다.
4. 확인되지 않은 서지정보나 숫자를 만들어내지 않는다.
