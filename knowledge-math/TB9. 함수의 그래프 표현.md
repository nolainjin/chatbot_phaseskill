### 가. 직관적 그래프 접근(점을 직접 찍기) 후 논리적 그래프 접근

#### 1) 특별한 형태를 의미하는 함수식(반드시 '모든 실수(특정 구간에서) $x$에 대해'라는 표현 포함)^85
[^85]: 모든 실수(특정 구간에서)라는 말이 없다면, 방정식일 수 있다.
만약, 모르는 표현 등장시, 반드시 점을 찍어 직관적으로 접근한 후 의미 파악^86
[^86]: 가)~바)까지의 표현식은 반드시 기억하고 있되, 단순히 암기만이 아니라 왜 해당식이 주어졌을 때 특별한 그래프적인 성질을 갖는지 명확하게 이해하고 있어야한다. 새로운 표현이 등장했을 때, 당황하지 않고, 바로 적응할 수 있도록 준비하자.

##### 가) $f(-x) = f(x)$ : 우함수 → $y$축$(x=0)$ 대칭인 함수의 그래프 작성

미분시 기함수, 적분시 기함수(단, 적분상수 $=0$일 때)

##### 나) $f(-x) = -f(x)$ : 기함수 → 원점$(0,0)$ 대칭인 함수의 그래프 작성

미분시 우함수, 적분시 우함수

##### 다) $f(a-x) = f(a+x)$ → $x=a$대칭인 함수의 그래프 작성^87

[^87]: 가)의 일반적인 형태. 함수 안의 $x$좌표의 합이 $2a$로 일정하다.

##### 라) $f(a-x) + f(a+x) = 2b$ : $(a,b)$ 대칭인 함수의 그래프 작성^88

[^88]: 나)의 일반적인 형태. 함수 안의 $x$좌표의 합이 $2a$로 일정하고, 함숫값의 합이 $2b$로 일정하다.

##### 마) $f(x) = f(x+p)$ : 주기가 $\frac{p}{n}$($n$은 자연수)인 함수의 그래프 작성^89

[^89]: 주기가 $n$이라고 착각하기 쉽다. 예를 들어, 주기가 $2$인 함수, 주기가 $3$인 함수 모두 모든 실수 $x$에 대하여 $f(x) = f(x+6)$을 만족할 수 있다.

##### 바) $f(x) = f(x-m) + n$ : 주기성을 가지며 평행이동^90

[^90]: 함수의 정의가 수열의 점화식처럼 주어질 때도 있다. 예를 들어, $1 \le x < 2$에 대해 $f(x)$의 식이 주어질 때, $f(x+1) = f(x) + 1$을 통해 실수 전체 구간에 대해 함수 $f(x)$를 추론할 수 있다.

#### 2) 직관적 그래프 접근법^91

[^91]: 미분을 하지 않아도, 직관적으로 그래프를 추측할 수 있다. 그래프의 개형을 추측하여 풀이의 방향성을 예측하는 것이 중요하다.

##### 가) $y = kf(x)$ → $y$값 $k$배

##### 나) $y = f(kx)$ → $y$축 기준으로 $x$축을 $\frac{1}{k}$배 축소, 확대

##### 다) $y = f(x) + g(x)$ 
→ 함숫값의 누적
→ $y = \frac{f(x)+g(x)}{2}$ (중점연결)의 2배
→ $y = f(x) - (-g(x))$^92

[^92]: $y = f(x) + g(x)$의 부호를 확인할 때, $f(x) - (-g(x))$를 통해 $y = f(x)$와 $y = -g(x)$ 교점의 좌우에서 부호변화를 좀 더 쉽게 확인할 수 있다.

##### 라) $y = f(x) - g(x)$ → 함숫값의 차이 또는 $y = \frac{f(x)+(-g(x))}{2}$ (중점연결)의 2배

##### 마) $y = f(x)g(x)$ → 부호에 따른 변화 관찰^93

[^93]: $y = f(x)$와 $y = g(x)$가 각각 $x$축과 만나는 지점에서 $y = f(x)g(x)$의 부호 변화가 생길 수 있다.

##### 바) $y = \frac{1}{f(x)}$ → $y=1, y=-1$을 기준으로 함숫값 역전

##### 사) $y = \frac{f(x)}{g(x)}$ → $y = f(x) \times \frac{1}{g(x)}$로 접근

#### 3) 논리적 그래프 접근 순서^94

[^94]: 가) ~ 사) 까지의 모든 절차를 다 거쳐야 하는 것은 아니다. 문제에서 주어진 조건과 구해야하는 대상에 필요한 단계만을 확인하면 충분하다. 앞 장에서의 직관적 그래프 접근을 통해 풀이의 방향성을 예측하고, 논리적 그래프 접근을 통해 정확하게 추론한다.

##### 가) 정의역(가능한 경우 치역도 확인)

특히 치역을 확인해야하는 함수 : $\sqrt{f(x)}$, $\frac{1}{f(x)}$, $\log_a f(x)$, $\tan f(x)$

##### 나) 대칭성, 주기성 확인

##### 다) 특정점(기준점 또는 절편) 확인

기준점이 있는 함수 : 이차함수(꼭짓점), 유리함수(점근선의 교점), 무리함수(꼭짓점), 지수함수$((0,1))$, 로그함수$((1,0))$

##### 라) 점근선 확인

점근선이 있는 함수 : 유리함수, 지수함수, 로그함수, $\tan$함수

##### 마) 증가, 감소 → 도함수의 부호

##### 바) 함숫값의 최대·최소 → $f'(x) = 0$^95

[^95]: 미분가능한 함수 $f(x)$에 대하여 $f'(x) = 0$인 $x$에 대하여 함수값이 최대 또는 최소일 수 있다. 다시말해 $f'(x) = 0$인 $x$가 함수값의 최대·최소 후보이다.

##### 사) 오목, 볼록 → 이계도함수의 부호

#### 4) 기울기값의 최대·최소 → $f''(x) = 0$^96

[^96]: 이계도함수를 갖는 함수 $f(x)$에 대하여 $f''(x) = 0$인 $x$에 대하여 기울기가 최대 또는 최소일 수 있다. 다시말해 $f''(x) = 0$인 $x$가 기울기 최대·최소 후보이다.

### 나. 함수 표현에 따른 함수 추론^97

[^97]: 이런 형태가 반드시 출제된다는 것은 아니다. 이러한 형식으로 주어졌을 때, 함수를 예측해 볼 수 있다는 뜻.

1) $f(x+y) = f(x) + f(y)$ → $f(x) = kx$

2) $f(x+y) = \frac{f(x)f(y)}{f(x)+f(y)}$ → $f(x) = \displaystyle\frac{k}{x}$

3) $f(x+y) = f(x)f(y)$ → $f(x) = a^x$

4) $f(xy) = f(x) + f(y)$ → $f(x) = \log_a x$

### 다. 일차함수 : 직선

1) 일차함수의 결정 : $y$축과 평행한 직선 위에 있지 않은 서로 다른 두점 또는 한 점과 기울기

2) 기울기의 값이 $m$으로 일정한 : $y = mx + k$ ($k$는 미정계수) → $k$에 따른 직선의 평행이동 관찰

3) 항상 $(a,b)$를 지나는 : $y = k(x-a) + b$ ($k$는 미정계수) → $k$에 따른 직선의 기울기 변화 관찰

4) $x$축과 평행한 또는 $y$축에 수직인 : $y = k$ → $k$에 따른 직선의 평행이동 관찰

5) $y$축과 평행한 또는 $x$축에 수직인 : $x = k$ → $k$에 따른 직선의 평행이동 관찰

6) $x$절편이 $a$, $y$절편이 $b$ : $\displaystyle \frac{x}{a} + \frac{y}{b} = 1$

7) 일반성 : $ax + by + c = 0$

8) $y = x$ → $x$좌표를 $y$좌표로, $y$좌표를 $x$좌표로 이동가능

### 라. 이차함수^98

[^98]: 무조건적으로 이차함수 식으로 $f(x) = ax^2 + bx + c$로 두는 학생들이 많다. 고난도 문제의 경우, 이렇게 식을 설정하였을 때 풀이가 조악해질 때도 많다. 따라서 문제의 조건에 따라 식을 선택하는 능력이 중요하다. 이차함수의 경우, 조건의 내용에 따라, 대표적으로 $f(x) = a(x-m)^2 + n$, $f(x) = a(x-\alpha)(x-\beta)$, $f(x) = ax^2 + bx + c$ 중 하나를 선택한다.

1) 이차함수의 결정 : 한 직선 위에 있지 않은 서로 다른 세점

2) 대칭성 → $f(x) = a(x-m)^2 + n$

3) $x$축과 접한다 → $f(x) = a(x-m)^2$

4) $x$축과의 교점 → $f(x) = a(x-\alpha)(x-\beta)$

5) 직선 $y = mx + n$과 접한다 → $f(x) - (mx+n) = a(x-p)^2$

6) 그 외 일반적 → $f(x) = ax^2 + bx + c$

### 마. 절댓값 기호가 있는 대표적 함수^99

[^99]: 절댓값기호가 있는 함수가 주어졌을 때, 크게 2가지 방식 중 하나를 선택한다. 첫 번째는 단계적 접근이다. $|f(x)|$, $f(|x|)$와 평행이동, 대칭이동하여 함수 그래프의 개형을 추론하는 방법, 두 번째는 압축된 표현의 전개이다. 대표적인 압축표현으로는 $|a-b|$,  $\sum$ ,  $\displaystyle\lim_{n \to \infty} x^n$, $\lbrack ~~~~~ \rbrack$등이 있다. 이러한 압축 표현이 나왔을 때, 기술적인 방법으로 접근이 힘들고, 이해가 어려울 때, 압축을 전개하면 풀이에 도움이 된다.

1) $y = |f(x)|$ → $x$축 아래 부분을 접어올림

2) $y = |f(x)-k|$ → $y = k$ 아래 부분을 접어올림(접어올린 선이 $x$축)

3) $y = |f(x)-k|+ k$ → $y = k$ 아래 부분을 접어올림

4) $y = f(|x|)$ → $x$축 우측 부분을 $y$축에 대칭, 우함수

### 바. 합성함수 → $g(f(x))$

#### 1) $g(f(x))$에 대해 $f$의 증감, 범위에 따라 $g$의 증감을 조사해 개형 작성,

| | $f$ 증가 | $f$ 감소 |
|---|---|---|
| $g$증가 | $g(f(x))$는 증가 | $g(f(x))$는 감소 |
| $g$감소 | $g(f(x))$는 감소 | $g(f(x))$는 증가 |

#### 2) $n$축 풀이^100

[^100]: $n$축 풀이의 경우 그래프 개형을 추축하기보다는, 극대, 극소의 개수를 추론하는데 적합하다.

### 사. 역함수 → 원함수를 $y=x$ 대칭,

- $f$가 증가함수일 때 역함수와의 교점은 모두 $y=x$위에 존재

- $f$가 감소함수일 때 역함수와의 교점 모두가 $y=x$위에 존재하는 것은 아님
  ($y=x$위에 있지 않은 교점은 $y=x$ 대칭쌍으로 존재)
