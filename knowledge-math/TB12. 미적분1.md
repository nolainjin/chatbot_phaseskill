
## 가. lim → 어떤 단원이든 lim가 있을 땐 무조건 꼴 가장 먼저 확인

### 1) 정형

### 2) 부정형
(무한대는 각각 커져가는 정도가 다를 수 있다. 무한소는 각각 0에 가까워져가는 정도가 다를 수 있다.)

#### 가) $\frac{\infty}{\infty}$
→ 분모에서 커져가는 정도가 가장 큰 항으로 분모, 분자를 나눈다.

#### 나) $\infty - \infty$
→ 커져가는 정도가 가장 큰 항으로 묶어낸다. 근호가 있을 때, 유리화

#### 다) $\frac{0}{0}$
① 영인자 약분
② 미분계수의 정의
③ 로피탈
④ 공식

#### 라) $0 \times \infty$
→ 변형(치환, 통분, 역수, 유리화 등)을 통해 $\frac{\infty}{\infty}$, $\frac{0}{0}$로 변형

### 3) 미정계수가 있는 $\frac{?}{0}$ 또는 $\frac{0}{?}$ 꼴

#### 가) $\frac{?}{0} = \alpha$
→ 분자가 극한값 0, 전체가 극한값 $\alpha$를 갖도록 식 2개 작성

#### 나) $\frac{0}{?} = \alpha$ 
→ 분모가 극한값 0, 전체가 극한값 $\alpha$를 갖도록 식 2개 작성

### 4) 도형에서의 극한
→ 문제에 표현된 도형 생성 순서를 통해 인과를 관찰하여 관계 도출, 기하적 분석

## 다. 연속

### 1) 함수가 $x = a$에서 연속

#### 가) $f(x) = \begin{cases} ~~~~~~ &(x\not=a) \\ ~~~~~~ &(x=a) \end{cases}$
→ 극한값=함숫값

#### 나) $f(x) = \begin{cases} ~~~~~~ & (x \geq a) \\ ~~~~~~ & (x < a) \end{cases}$
→ 좌극한(또는 우극한)=함숫값

### 2) $\begin{cases} f\text{가 } x = a\text{에서 불연속} \\ f\text{가 }x = a \text{근방에서 유한한 함숫값} \\ g\text{가 } x = a\text{에서 연속} \\ f(x) g(x)\text{가 } x = a\text{에서 연속}\end{cases}$→ $g(a) = 0$


### 3) $f(g(x))$는 실수 전체에서 연속
→ $g$가 불연속인 $x$값과 $f$가 불연속인 $x$값 $k$에 대해 $g(x) = k$인 $x$에서 연속 여부 조사

## 라. 미분

### 1) $f$가 $x = a$에서 미분가능
① $f'(a)$, 즉 $\displaystyle\lim_{x \to a} \frac{f(x) - f(a)}{x - a}$가 존재
② $f$가 $x = a$에서 연속, 좌미분계수=우미분계수

### 2) $f$가 $x = a$에서 미분불가능
① $f'(a)$, 즉 $\displaystyle\lim_{x \to a} \frac{f(x) - f(a)}{x - a}$가 존재하지 않음 (좌미분계수≠우미분계수 또는 발산)
② $f$가 $x = a$에서 불연속 또는 첨점

### 3) $g(x)=|f(x)|$가 $g(a) = 0$이고 $x=a$에서 미분가능
→ $y = f(x)$가 $x$축을 접하면서 뚫는다/튕긴다.

### 4) $g(x) = |f(x)-(mx+n)|$가 $g(a) = 0$이고, $x=a$에서  미분가능
→ $y = f(x)$이 $y = mx+n$을 접하면서 뚫는다/튕긴다. (변곡접선과 관련)

### 5) $f(x+y)$ + 미분계수
→ 미분계수의 정의 또는 편미분

### 6) 접선
① 접점의 좌표가 주어졌을 때
② 접점에서의 기울기가 주어졌을 때
③ 외부의 한점이 주어졌을 때

### 7) 공통접선
→ $f(a) = g(a)$, $f'(a) = g'(a)$

### 8) 접선의 개수
→ 접점의 개수(접점의 $x$좌표를 $t$라고 했을 때, $t$에 관한 방정식의 해의 개수 → 방정식을 그래프로 표현했을 때의 교점의 개수)

### 9) 기울기의 최솟값
→ 변곡접선의 기울기 (원래 함수와 도함수의 관계를 이해할 것)

### 10) 다항함수의 미분방정식
① 다항함수 구하기
② 차수 결정 후 계수 결정
③ $f(x) = x^n g(x)$으로 두어 차수와 최고차항 계수 동시 결정

### 11) 그래프를 이용한 함수식의 추론

#### 가) 삼차함수

##### (1) $x$축과 한 점에서 접한다
→ $f(x) = a(x - p)^2(x - q)$

##### (2) $x$축을 접하면서 뚫는다
→ $f(x) = a(x - p)^3$

##### (3) $x$축과의 교점
→ $f(x) = a(x - p)(x - q)(x - r)$

##### (4) 직선 $y = kx + n$과 한점에서 접한다
→ $f(x)-(kx + n) = a(x - p)^2(x - q)$

##### (5) 직선 $y = kx + n$을 접하면서 뚫는다
→ $f(x)-(kx + n) = a(x - p)^3$

##### (6) 변곡점 $(p, q)$ 대칭성
→ $f(x) = a(x - p)^3 + b(x-p)+q$

##### (7) 일반성
→ $f(x) = ax^3 + bx^2 + cx + d$ 또는 $f'(x) = ax^{2}+bx+c$

##### (8) 극값과 변곡점을 지나는 직선과의 교점 사이의 길이비
→ $1 : \sqrt3$

##### (9) 접점과 변곡점, 접점을 지나는 직선과의 교점 사이의 길이비
→ $1 : 1 : 1$

---

#### 나) 사차함수

##### (1) $x$축과 한 점에서 접한다
→ $f(x) = a(x - p)^2(x - q)(x - r)$

##### (2) $x$축과 두 점에서 접한다
→ $f(x) = a(x - p)^2(x - q)^2$, 대칭성 이용 가능

##### (3) $x$축을 접하면서 뚫는다
→ $f(x) = a(x - p)^3(x - q)$

##### (4) $x$축과의 교점
→ $f(x) = a(x - p)(x - q)(x - r)(x - s)$

##### (5) 직선 $y = kx + n$과 두 점에서 접한다
→ $f(x) - (kx + n) = a(x - p)^2(x - q)^2$

##### (6) 직선 $y = kx + n$을 접하면서 뚫는다
→ $f(x) - (kx + n) = a(x - p)^3(x - q)$

##### (7) 일반성
→ $f(x) = ax^4 + bx^3 + cx^2 + dx + e$ (이렇게 설정하는 경우는 거의 없다)  
또는 $f'(x) = ax^{3}+bx^{2}+cx+d$

## 마. 적분

### 1) 정적분의 계산
→ 구간동일, 피적분함수 동일, 구간에 따라 정의된 함수, 대칭성, 주기성

### 2) 적분 방정식

#### 가) 구간 상수 → 정적분 상수 취급
피적분함수에 적분변수외의 변수 있을 시 상수인 적분 구간이 변수로 바뀔 수 있음

#### 나) 구간 변수 → 정적분 함수 취급
대입과 미분 가능, 피적분함수에 적분변수외의 변수 있을 시 적분변수 외 변수 반드시 분리(전개 또는 치환 등)

피적분함수에 적분변수외의 변수는 적분 내에 있을 때, 상수 취급

### 3) 정적분으로 정의된 함수 $\displaystyle\int_{a}^{x}f(t)dt$
→ $g(x) = \displaystyle\int_a^x f(t)dt$로 도함수를 $f$로 하는 함수 취급  
($g'(x) = f(x)$의 부호를 통해 $y = g(x)$ 그래프 작성 가능)

### 4) 정적분으로 정의된 함수의 최대·최소
→ ① 식의 최대·최소 ② 넓이의 최대·최소

### 5) $\lim$ 와 정적분
→ 미적분학의 기본 정리 $\displaystyle\int_{a}^{b}f(x)dx=F(b)-F(a)$ 이용

### 6) 정적분을 이용한 넓이의 계산 → 그래프로 표현 후 정적분 계산 고려
#### 가) 직선과 이차함수로 둘러싸인 도형의 넓이
→ $\displaystyle\frac{|a|(\beta-\alpha)^{3}}{6}$
#### 나) 서로 다른 이차함수로 둘러싸인 도형의 넓이
→ $\displaystyle\frac{|a-a'|(\beta-\alpha)^{3}}{6}$

### 7) 수직선에서의 운동(반드시 $t=0$ 일때의 위치를 주의해야한다.) → 위치, 속도, 가속도 간의 미분 수준 이해 필수

#### 가) 위치 → 속도$v$의 정적분( $v-t$그래프에서의 수학적 넓이)  
#### 나) 움직인 거리 →  $|\text{속도}v|$의 정적분( $v-t$ 그래프에서의 실제 넓이)
